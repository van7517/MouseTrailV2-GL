# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.2] - 2026-07-24

### 新增

- 系统托盘后台运行（无控制台、不占任务栏）
- 中文设置界面（可视化调节参数）
- 进程黑名单点选添加
- 开机自启选项
- 发布包仅单个 `MouseTrailV2.exe`

### 改进

- 细软幽灵描边与更自然的变细消失
- 默认残留 **800 ms**
- 缩放 Scale 范围 **1～10**
- 配置写入 `%AppData%\MouseTrailV2\`

## [0.1.1] - 2026-07-23

### 改进

- 连续丝带绘制与预乘 Alpha
- 白底观感优化

## [0.1.0] - 2026-07-23

### 新增

- Windows 透明叠加层彩虹鼠标轨迹
- OpenGL 渲染与基础配置
- GitHub Actions 自动构建
