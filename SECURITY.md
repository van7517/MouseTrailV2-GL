# 安全策略

## 支持的版本

当前仅维护仓库 `main` 分支上的最新发布版本。

| 版本 | 支持 |
|------|------|
| latest release | ✅ |
| 历史版本 | ❌ |

## 报告漏洞

如果你发现安全相关问题（例如异常权限请求、意外的网络行为、可被利用的崩溃等），请**不要**公开开 Issue 讨论细节。

请通过以下方式私下联系维护者：

- GitHub：[@van7517](https://github.com/van7517) 私信 / Security advisory（若已启用）

请尽量包含：

1. 问题描述与影响范围
2. 复现步骤
3. 受影响版本 / commit
4. 可行的修复建议（如有）

我们会尽快确认并回复。在修复发布前，请勿公开完整利用细节。

## 说明

本项目为本地桌面叠加层工具，默认不访问网络。若未来引入联网功能，会在 README 与更新说明中单独标注。
