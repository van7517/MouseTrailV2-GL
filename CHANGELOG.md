# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.2] - 2026-07-24

### 改进

- 细软幽灵描边
- 颜色更淡、默认更细
- 残留更长，消失更慢

## [0.1.1] - 2026-07-23

### 改进

- 连续三角带丝带（非圆点/线段）
- 尾部以**逐渐变细**方式消失，避免突然截断
- 预乘 Alpha，减轻白底白边
- 默认更粗、白底更清晰

## [0.1.0] - 2026-07-23

### 新增

- Windows 全屏透明叠加层鼠标轨迹
- 彩虹幽灵缎带效果（OpenGL）
- `config.json` 配置：透明度、粗细、粒子、黑名单、热键等
- `F8` 开关 / `F9` 退出
- GitHub Actions 自动构建 Windows x64 发布包
